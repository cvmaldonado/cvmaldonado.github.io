Fundamentos teóricos

Conceptos aplicados
en este proyecto

⚡
01 · ELECTRÓNICA BÁSICA

Sensores Analógicos

Un sensor convierte una magnitud física (temperatura) en una señal eléctrica medible. El LM35 genera un voltaje proporcional a la temperatura: exactamente 10 mV por cada grado Celsius.

🔄
02 · CONVERSIÓN

Analógico → Digital

El Arduino Nano tiene un convertidor ADC de 10 bits que transforma el voltaje continuo del sensor en un número entero entre 0 y 1023, procesable por el microcontrolador.

🖥️
03 · MICROCONTROLADORES

Arduino Nano

Plataforma de desarrollo basada en el ATmega328P. Combina CPU, memoria Flash y periféricos en un chip. Opera a 16 MHz y es totalmente compatible con el sensor LM35 y la pantalla LCD I2C.

📡
04 · COMUNICACIÓN

Protocolo I2C

Estándar de comunicación serial síncrona que usa solo 2 líneas (SDA y SCL). Permite conectar la pantalla LCD usando únicamente los pines A4 y A5 del Nano, simplificando enormemente el cableado.

🔬
05 · SEMICONDUCTORES

Diodos y Transistores

El LM35 usa internamente transistores BJT en configuración diferencial para lograr la respuesta lineal. Los diodos protegen el circuito contra inversiones de polaridad accidentales.

💻
06 · PROGRAMACIÓN

Lectura y Control

Con analogRead() se captura el valor ADC, se convierte a temperatura con la fórmula del LM35 y se muestra en pantalla con lógica condicional que activa alertas según el umbral configurado.

T(°C) = Vout ÷ 10

donde Vout está en milivoltios
Ejemplo: 300 mV → 30.0 °C
Ejemplo: 250 mV → 25.0 °C
Ejemplo: 375 mV → 37.5 °C

La fórmula del LM35

El LM35 es un sensor de temperatura calibrado de fábrica que entrega exactamente 10 mV por cada grado Celsius. No requiere calibración externa.

En el código de Arduino, la conversión completa es:

01
Leer el valor ADC con analogRead(A0) → valor entre 0 y 1023
02
Convertir a voltaje: voltaje = valor × (5.0 / 1023.0)
03
Obtener temperatura: temp = voltaje × 100.0

Implementación

Código fuente

termometro.ino ARDUINO C++ 
// Termómetro Digital con Arduino Nano + LM35 + LCD I2C
// Electrónica Analógica · 2025

#include <Wire.h>
#include <LiquidCrystal_I2C.h>

LiquidCrystal_I2C lcd(0x27, 16, 2);  // Dirección I2C, cols, filas
const int sensorPin = A0;

void setup() {
  lcd.init();
  lcd.backlight();
  lcd.setCursor(0, 0);
  lcd.print("  Iniciando...");
  delay(1500);
}

void loop() {
  int lectura    = analogRead(sensorPin);
  float voltaje  = lectura * (5.0 / 1023.0);
  float temp     = voltaje * 100.0;

  lcd.clear();
  lcd.setCursor(0, 0);

  if (temp < 30.0) {
    lcd.print("TEMP NORMAL");
  } else if (temp < 38.0) {
    lcd.print("TEMP ELEVADA");
  } else {
    lcd.print("ALTA TEMP !");
  }

  lcd.setCursor(0, 1);
  lcd.print(temp, 1);
  lcd.print(" C");

  delay(1000);  // Actualizar cada segundo
}
